# 项目阶梯

## 可以立即完成的项目

- [项目 1：端到端表格分类教程](01-tabular-classification.md)
- [项目 1 完整可运行代码](../scripts/project_tabular_classification.py)

| 级别 | 项目 | 核心能力 | 最低验收标准 |
|---|---|---|---|
| 1 | 学习时长与分数预测 | 回归、损失、梯度下降 | 从零实现并画出拟合直线 |
| 2 | Titanic 生存预测 | 清洗、分类、交叉验证 | 有基线、Pipeline、指标和错误分析 |
| 3 | 客户分群 | 标准化、K-Means、PCA | 解释每个群体而非只画图 |
| 4 | FashionMNIST 分类 | PyTorch 训练循环、CNN | 固定随机种子，保存最佳模型 |
| 5 | 猫狗迁移学习 | 数据增强、迁移学习 | 比较冻结与微调，记录实验 |
| 6 | 文本分类 | token、embedding、序列模型 | 与传统 TF-IDF 基线比较 |
| 7 | 开源复现与改进 | 读代码、复现、消融实验 | README 可复现，报告改动和局限 |

每个项目必须包含：问题定义、数据来源与许可、数据划分、基线、评价指标、实验配置、结果、错误分析、局限和复现命令。
