# 4.1｜神经网络：从线性模型到多层感知机

## 1. 为什么需要隐藏层

单个线性模型只能画一个超平面，无法解决 XOR 这类非线性问题。多层感知机在两次线性变换之间加入非线性激活：

```text
h = tanh(XW₁+b₁)
z = hW₂+b₂
p = sigmoid(z)
```

如果去掉激活函数，多层线性变换仍可合并成一次线性变换，深度没有意义。

## 2. 参数和形状

若输入有 2 个特征、隐藏层 8 个单元、输出 1 个概率：

```text
X:  (n,2)
W₁: (2,8)   b₁: (1,8)
h:  (n,8)
W₂: (8,1)   b₂: (1,1)
p:  (n,1)
```

每写一层先检查 shape。批大小 n 可以变化，参数形状不能随批大小变化。

## 3. 激活函数

- ReLU：`max(0,x)`，计算简单，是隐藏层常用默认。
- tanh：输出 `[-1,1]`，小模型和循环网络中常见。
- sigmoid：输出 `[0,1]`，适合二分类输出，但深层隐藏层中容易梯度变小。

多分类输出常用 softmax，使每一行类别概率和为 1。

## 4. 损失函数与输出必须匹配

- 回归：线性输出 + MSE/MAE。
- 二分类：一个 logit + 二元交叉熵。
- 多分类：C 个 logits + 交叉熵。

logit 是进入 sigmoid/softmax 前的任意实数。成熟框架通常把激活与交叉熵合并计算以提高数值稳定性，因此不要重复 sigmoid。

## 5. 容量、欠拟合与过拟合

隐藏层更宽、更深会增加表达能力，也更容易过拟合。先比较训练与验证曲线：两者都差是欠拟合；训练好、验证差是过拟合。常见手段包括更多数据、数据增强、权重衰减、Dropout、早停和减小模型。
