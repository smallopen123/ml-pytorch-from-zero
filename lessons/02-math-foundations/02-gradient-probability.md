# 2.2｜导数、梯度、链式法则与概率

## 1. 导数回答“参数动一点，损失怎么变”

对单参数函数 `f(w)`，导数近似为：

```text
f'(w) ≈ [f(w+ε)-f(w-ε)] / (2ε)
```

多个参数的所有偏导数组成梯度。梯度指向函数增长最快的方向，所以梯度下降使用相反方向：

```text
参数 ← 参数 - 学习率 × 梯度
```

学习率太小会很慢，太大会越过最低点甚至发散。

## 2. 均方误差梯度

线性模型 `ŷ_i = wx_i`，均方误差：

```text
L = (1/n) Σ(ŷ_i-y_i)²
dL/dw = (2/n) Σ(ŷ_i-y_i)x_i
```

这里用到了链式法则：损失依赖预测，预测又依赖权重；局部导数相乘把影响传回权重。神经网络反向传播就是高效、系统地重复链式法则。

## 3. 数值梯度检查

解析梯度来自推导和代码，数值梯度来自有限差分。两者接近说明反向传播很可能正确。运行：

```powershell
python scripts/math_gradient_demo.py
```

数值梯度很慢，只适合检查小模型，不用于正式训练。

## 4. 概率与分类

分类模型常输出 `P(y=1|x)`。概率必须在 0 到 1 之间，逻辑回归用 sigmoid：

```text
σ(z) = 1 / (1 + exp(-z))
```

概率不是类别。阈值 0.5 只是默认决策规则；疾病筛查可能降低阈值提高召回率，代价是更多假阳性。

## 5. 期望、方差与抽样

期望描述长期平均，方差描述波动。训练/验证指标也会因抽样而变化，所以应固定随机种子、使用交叉验证，并报告均值与波动，而不是只挑最好的一次。
