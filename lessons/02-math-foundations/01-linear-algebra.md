# 2.1｜线性代数：把一批数据同时送进模型

## 1. 标量、向量、矩阵

- 标量：一个数，例如学习率 `0.01`。
- 向量：一组有顺序的数，例如一个样本 `[面积, 房龄, 距离]`。
- 矩阵：多行向量组成的数据集；`X ∈ R^(n×d)` 表示 n 个样本、d 个特征。

## 2. 点积就是加权求和

若 `x=[2,3]`、`w=[0.5, 4]`，则 `x·w = 2×0.5 + 3×4 = 13`。线性模型对一个样本的预测是：

```text
ŷ = x·w + b
```

对所有样本同时预测：

```text
ŷ = Xw + b
X: (n,d), w: (d,), ŷ: (n,)
```

矩阵乘法要求左侧的列数等于右侧的行数。做题时先写 shape，能解决大多数维度错误。

## 3. 转置为什么出现在梯度里

`X.T` 把 `(n,d)` 变成 `(d,n)`。误差向量是 `(n,)`，所以 `X.T @ error` 得到 `(d,)`：每个特征对应一个权重梯度。

## 4. 范数与距离

L2 范数 `||x||₂ = sqrt(sum(x_i²))` 表示向量长度；两个样本的欧氏距离是 `||x_a-x_b||₂`。KNN 和 K-Means 依赖距离，所以不同量纲的特征通常必须标准化。

## 5. 练习

3. 身高用厘米、收入用元时，哪一列更容易主导欧氏距离？如何修复？
4. 用 NumPy 验证 `X @ w` 等于逐行点积。
